See It In Action

Hello World

task "Hello World" on local:
    run "echo Hello from SimpleInfra!"
    run "echo Current directory:"
    run "cd"

Web Server Setup

task "Setup Web Server" on web:
    install nginx
    install python3

    start service nginx
    ensure port 80 is open

    run "echo Server ready!"

With Variables

set app_name "myapp"
set version "1.0.0"

task "Deploy" on servers:
    run "echo Deploying {app_name} v{version}"
    create directory "/opt/{app_name}"

IoT Automation

task "Setup IoT Gateway" on pi:
    raspberrypi:
        action "setup_iot_gateway"
        install_mqtt "true"
        install_nodered "true"

SimpleInfra vs Ansible

Feature SimpleInfra Ansible
Learning Curve Minutes Hours to Days
Syntax English-like DSL YAML (verbose)
Execution Speed Async (Fast) Sync (Slower)
IoT Support Built-in (Raspberry Pi, Arduino) Limited
Network Security 8 modules (Illumio-style) Basic
Windows Compatible Full support WSL required
File Size Small Large (lots of boilerplate)
